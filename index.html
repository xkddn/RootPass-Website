[ 01 ] — fonctionnalités

Conçu pour les paranoïaques.
Pensé pour le quotidien.

Toutes les garanties d'un coffre-fort chiffré, sans aucun compromis sur le confort.

01

Chiffrement inviolable

Votre coffre est protégé par AES-256-GCM, le même standard que les agences gouvernementales. La clé est dérivée à la volée à partir de votre Master Password - elle n'est jamais stockée, nulle part.

AES-256-GCM · clé dérivée au vol

02

Le cloud vous appartient

Synchronisez via votre propre OneDrive, Dropbox ou une simple clé USB. Zéro serveur central, vos données ne quittent jamais votre contrôle.

03

2FA natif

Générez vos codes TOTP directement dans l'application. Plus besoin d'une app tierce pour votre double authentification.

04

Recherche Spotlight

Un raccourci global pour tout retrouver en une seconde, sans même ouvrir la fenêtre principale. Tapez, trouvez, collez.

Ctrl + Shift + Space

[ 02 ] — installation

Télécharger en toute confiance

Open source, builds publics et vérifiables. Voici comment installer en quelques secondes.

Windows (.exe) Linux (AppImage / .deb) macOS : Encore en dev, bientôt !

« Éditeur inconnu » au lancement ? C'est normal.
RootPass n'est pas signé avec un certificat payant : votre système affiche donc un avertissement, comme pour la plupart des apps open source. Le code et les builds étant entièrement publics, vous pouvez vérifier vous-même qu'il n'y a rien à cacher.

Suivez le guide de "Transparence et vérification" ci-dessous pour être sûr à 100 % de l'intégrité de votre téléchargement. Si vous avez la moindre question, n'hésitez pas à ouvrir une issue sur GitHub.

Important : Sauvegardez votre Master Password !!!!
Si vous le perdez, le coffre est IRRÉCUPÉRABLE. Aucune réinitialisation n'est possible, même par moi. C'est le prix du chiffrement de bout en bout. Soyez intelligent·e, notez-le dans un endroit sûr et tout ira bien.

Installation rapide

Windows

  1. Lancez le fichier .exe.
  2. Sur l'écran SmartScreen, cliquez sur Informations complémentaires.
  3. Cliquez sur Exécuter quand même.

macOS

  1. Ouvrez le fichier .dmg.
  2. Faites un clic droit sur l'application puis Ouvrir.
  3. Confirmez l'ouverture dans la fenêtre qui apparaît.

Linux

Rendez l'AppImage exécutable, ou installez le paquet .deb :

chmod +x RootPass*.AppImage && ./RootPass*.AppImage

Transparent et vérifiable

Optionnel ! Seulement si tu veux vérifier l'intégrité de l'app

RootPass est 100 % open source. Chaque installeur est compilé automatiquement par GitHub Actions à partir du code public, rien n'est ajouté à la main, tout est traçable.

Windows

Vérifiez l'intégrité : lancez cette commande dans PowerShell, puis comparez à l'empreinte officielle.

certutil -hashfile <fichier> SHA256
