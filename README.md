# RootPass — Site Vitrine Officiel

Ce dépôt contient le code source de la landing page officielle de **RootPass**, le gestionnaire de mots de passe 100 % local et respectueux de la vie privée.

🌐 **Site internet officiel :** [https://root-pass-website.vercel.app/](https://root-pass-website.vercel.app/)

---

## 🚀 Présentation

Une (SPA) épurée, rapide et entièrement optimisée pour le SEO, conçue pour présenter l'application RootPass et distribuer ses versions officielles de manière transparente.

### Fonctionnalités du site :
- **Optimisation SEO & Accessibilité :** Structure HTML5 sémantique, balisage Open Graph complet, JSON-LD (Schema.org) et sitemap natif.
- **Performance :** Temps de chargement instantané grâce à une architecture statique sans framework lourd.
- **Design System :** Reprise fidèle de la charte graphique de l'application (Mode sombre profond `#08080A`, accents Emerald, typographie Poppins et effets de lueur subtils).
- **Distribution Sécurisée :** Liens directs vers les releases officielles compilées par GitHub Actions.

## 🛠️ Stack Technique

- **Build Tool :** [Vite](https://vite.dev/)
- **Structure :** HTML5 Sémantique
- **Styles :** [Tailwind CSS v4](https://tailwindcss.com/)
- **Icônes :** [Lucide Icons](https://lucide.dev/)
- **Hébergement :** [Vercel](https://vercel.com/)
